API 檢索

All API calls start with:
http://itswindtw.info:9001/api/

User

List Users

GET users/
Example request
curl http://itswindtw.info:9001/api/users/
[
    {
        id: 1,
        name: null,
        username: "skuld",
        avatar: {
            medium: "/images/avatar_missing.jpg",
            thumb: "/images/avatar_missing.jpg"
        }
    },
    {
        id: 2,
        name: "user1",
        username: "user1",
        avatar: {
            medium: "/images/avatar_missing.jpg",
            thumb: "/images/avatar_missing.jpg"
        }
    }
]

Get a User

GET users/:id
Example request
curl http://itswindtw.info:9001/api/users/1
[
    {
        id: 1,
        name: null,
        username: "skuld",
        avatar: {
            medium: "/images/avatar_missing.jpg",
            thumb: "/images/avatar_missing.jpg"
        }
    }
]

Status

GET /users/sign_in
Example request
curl http://itswindtw.info:9001/api/users/sign_in
null
{
    email: banana@mail.com,
    name: banana, 
    username: banana
}

Sign In

POST /users/sign_in
Parameter (stringified)
Example request
curl http://itswindtw.info:9001/api/users/sign_in -X POST
{
    success: true,
    auth_token: 123, 
    ...
}

Sign Out

DELETE /users/sign_out
Example request
curl http://itswindtw.info:9001/api/users/sign_out -X DELETE

Locale

Project

List Projects

GET projects/
Example request
curl http://itswindtw.info:9001/api/projects/
[
    {
        id: 1,
        info: "The New TryNeeds",
        name: "Skuld"
    }
]

Get a Project

GET projects/:id
Example request
curl http://itswindtw.info:9001/api/projects/1
{
    file_type: 0,
    id: 1,
    info: "The New TryNeeds",
    name: "Skuld",
    tags: [ ]
}
